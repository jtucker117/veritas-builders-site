**landscape 1536 × 1024**, save as **.jpg**, target **under 500 KB**, and drop it
in this folder (`images/projects/`) using the exact filename listed.

Style baseline for every prompt (already appended to each below):
> Photorealistic architectural photography, natural daylight, wide-angle but
> undistorted, professional real-estate/construction magazine quality, no people,
> no text, no watermarks, no logos, no visible brand names.

## Filing checklist

1. Save each generated image as **.jpg** at **1536 × 1024**.
2. Compress to **under 500 KB** (Preview → Export → Quality ~70, or `sips`).
3. Drop it into `images/projects/` with the **exact filename** above.
4. Wire it into the page:
   - **Group A:** replace `<article class="project placeholder">` with a real
     `<article class="project">` block (copy the working example directly above it
     in the same file) and delete the `<div class="project-photo" role="img" ...>`.
   - **Group B:** these pages have no gallery section yet — a new
     `<section class="section" id="work">` block needs to be added, modeled on
     `kitchen-remodel.html:144-178`.
5. Write real descriptive `alt` text (location + scope + finishes), matching the
   style of the existing tiles — it's doing SEO work on these pages.
